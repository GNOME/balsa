Introduction

Balsa is a graphical mail client similar to Eudora on Windows. It supports the standard features of any good reader and has new features added continually.

Balsa is written using GTK+ and Gnome. It is based on libmutt, the backend code to Mutt, but expands far beyond being a graphical Mutt. Balsa supports: IMAP and POP3 accounts; sending mail via sendmail or direct SMTP; optional multithreaded mail retrieval and sending; support for all configurations of local mail folders; automatic mail checking; mail printing; inline attachment viewing; and the proverbial much, much more.

If you want to learn more about Balsa, your best bet is the mailling list <balsa-list@gnome.org>, which can be subscribed to by sending a message with a subject of subscribe to <balsa-list-request@gnome.org>. Bugs can be reported via bug-buddy, gnome-bug or to <submit@bugs.gnome.org>.

Getting Started with Balsa

If you're starting Balsa for the first time, the first thing you'll see is a druid that will set up Balsa's basic settings. Balsa can guess most of what it needs to know about you. It will also create five standard local mailboxes: the Inbox, Outbox, Sentbox, Draftbox, and Trash. (These can be changed to be IMAP or POP3 mailboxes later.)

There are many more settings in Balsa's preferences window, but most of them do not need to be changed.

Once Balsa is configured, you will proceed on to the main window.

Balsa download

Requirements

  1. A basic GNOME installation.
  2. Balsa needs not libPropList any more (true for 0.9.5 and higher). If you needed it for an older version, check if you don't have it already on your system, subsequently try to download suitable rpm. If you cannot do it for some reason, grab libPropList.tar.gz, It must be installed with a prefix of /usr if you wish to also use a Balsa RPM. Note, that in this last case you might need to ignore dependices warnings when installing rpm.
  3. Balsa 0.9.5 and greater also uses the libpspell library for spell checking, which is available in both rpm and deb format as well as plain source. You can find the library and modules at the pspell website. The same comment above about using Balsa rpms with self-compliled libraries applies here as well.
  4. The binaries come with GSS authorization enabled. You need Kerberos libraries (krb5-libs and krb5-config) provided with RH-6.2 and up, for example. Alternatively, you can recompile balsa from the source (man rpm is your friend) with GSS disabled.

Download

balsa-1.0.pre2 released on 2000.10.06 is the second in the prerelease series. If you have any problems with earlier versions, upgrade. If you don't have any problems with earlier versions, upgrade, too.

balsa-1.0.pre2 binary rpm is compiled on RH-6.2 with update packages installed with gnome-print-0.20 (available from HelixCode). You can get rid of this dependency by recompiling balsa. You may want to recompile balsa to enable LDAP support or HTML support, which are disabled by default.

Experimental release - 0.9.5

2000.09.18: This is pretty much all we have in CVS: new message browser, improved address book handling, more reliable mailbox backend, alias expansion. We have also now GSS and CRAM-MD5 authorization for IMAP, and APOP for POP3. New 0.9.5 features include:

  • improved multi-threading.
  • new warning/error reporting code.
  • LDAP support.
  • spell check (pspell package required tu compile and use balsa).
  • gnome-config used for storing/reading configuration.
  • multiple address book support. vCard- and LDAP-types implemented.
  • manual updates
  • gnome-print used for printing.

You can download it from:

  • http://www.theochem.kth.se/~pawsa/balsa/balsa-0.9.5.tar.gz
  • http://www.theochem.kth.se/~pawsa/balsa/balsa-0.9.5-1.src.rpm
  • http://www.theochem.kth.se/~pawsa/balsa/balsa-0.9.5-1.i386.rpm

CVS source

Balsa source is also available from GNOME CVS repository. Check out balsa module. The CVS version is sometimes not sufficiently stable for everyday use it but has all the new features and is best starting point if you want to hack balsa.

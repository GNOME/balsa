Balsa is an e-mail client for GNOME. With an interface similar to Eudora's, Balsa is highly configurable and has all the features you would expect in a robust mail client.

News

2002-01-20
balsa-1.2.4 is released. This release includes few bug fixes with respect to previous stable release: mailbox synchronize opereration is improved, IMAP passwords do not have to be stored any more in MT build, message threads are traversed more consistently, and few more.
2001-12-16
Development balsa-1.3.0 is released: first step to searching and filters; require gnome-1.4 or newer: gdk_pixbuf, gnome-vfs and some other; delayed IMAP tree scanning; "reply to group" functional.
2001-11-05
balsa-1.2.3 is released. Few bugs has been fixed and some more rough UI edges polished.
2001-10-31
balsa-1.2.2 is released. Few bugs has been fixed, rough UI edges polished and translation updated. (see NEWS or Changelog).
2001-10-20
balsa-1.2.1 is released, This release includes a load of new features, including new message forwarding functions, handling format=flowed, extended D&D support, new toolbar items, and many, many more (see NEWS or Changelog).
2001-09-19
Secretly. Silently. Certainly. Yes, it is time for balsa-1.2.0 release. There are no major changes compared to 1.2.pre3 but we have armoured the code against the unexpected.
2001-08-31
balsa-1.2.pre3 includes many fixes and has been tested on a larger variety of platforms. This is most likely last release between 1.2.0. Anyway, if you had some problems with 1.2.pre2 (or older), you should try this one to make sure that your problem has been solved.
2001-08-23
balsa-1.2.pre1 turned out to be pretty stable but we managed to nail down some extra problems and annoyances. Download balsa-1.2.pre2 from the usual place.
2001-08-08
it is time for a prerelease to next major version. You can get balsa-1.2.pre1 in the usual place.

Balsa is an e-mail client for GNOME. With an interface similar to Eudora's, Balsa is highly configurable and has all the features you would expect in a robust mail client.

GNOME2 port of balsa - second preview - is released. It is mostly work behind the scenes but the outcome is definetely visible.

2002-05-15

Development balsa-1.3.6 is released. It can search with reasonable speed in message bodies held in IMAP mailboxes. It has few portability fixes for Solaris. It should not crash any more when called with -a option. It has minor filter UI improvements (widgets are sensitive). It is more verbose on common SMTP errore and again does RFC-compliant header encoding. Some build fixes has been resolved as well.

2002-04-24

Development balsa-1.3.5 is released - observer though that we approach a stable phase with this branch. Major changes are: New mailboxes scanned in a thread. Improved message include function. It preserves "References" headers over message postpone. Number of crashes and other bugs fixed. Some portability fixes.

We release also balsa-2.0.0 technology preview. It is a GNOME2 port of balsa.
